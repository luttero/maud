Introduction

Here are the instructions to perform a quantitative analysis on a ZrO2/Al2O3 powder sample. We suppose the user has collected a spectrum using an x-ray diffractometer.The procedure will start creating a basic analysis file setting up the instrument, the dataset, the phases and the sample.We will build this analysis file in four steps:

  • Step 1: create and define the instrument
  • Step 2: create dataset and load the datafile
  • Step 3: load and modify the phases
  • Step 4: define the sample

Once defined the analysis file, we will start the analysis following two steps:

  • Step 5: set up some "visible" parameters (specifically, the T-PSZ cell parameters)
  • Step 6: select the iterations number and refine the spectrum

The final step (Step 7) will show the analysis output to the user. The tutorial will also guide the user to a more accurate and precise analysis if some parameters were not as expected.

The tutorial has been added by several figures to help the novice in figuring out easily the basic procedure to perform this kind of analysis.
